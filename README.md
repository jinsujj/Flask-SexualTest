## 핵심 기능  Key Feature
- 각 문항을 버튼형태로 + CSS 애니메이션 
- MongoDB 로 응답한 문항 및 정보 Flask 서버로 전송 
- Flask 서버에서 각 조건에 따른 결과 page return
